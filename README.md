# Crypto Trading Swarm

Phase `0.1` repository setup for a multi-agent crypto trading system.

## Core Project Docs

- Architecture and build direction: `cursor_docs/crypto_swarm_dev_doc.md`
- Coding standards and implementation rules: `cursor_docs/crypto_swarm_coding_practices.md`
- Phase and gate checklist: `cursor_docs/crypto_swarm_build_checklist.md`

## Current Scope

This repo currently provides:

- Initial project skeleton
- Environment placeholder setup
- Configuration scaffolding
- Infrastructure bridge files
- Base local container and test scaffolding

## Quick Start

1. Copy `.env.example` to `.env` if needed.
2. Fill all API and service placeholders in `.env`.
3. Install dependencies with Poetry:
   - `poetry install`
4. Run tests:
   - `poetry run pytest`
